## 🚀 三种使用方式

### 方式1：元Prompt库（推荐新手）⭐⭐⭐

**5分钟快速体验**：

1. 打开 `04-元Prompt库/快速启动/首次协作Prompt.md`
2. 复制其中的Prompt模板
3. 粘贴到AI对话框，填写参数
4. AI自动按DDAC规范执行

**17个Prompt分类**：
- 快速启动（3个）：首次协作、继续任务、快速整理
- 规范建立（3个）：建立规范、优化规范、一致性检查
- 文档处理（4个）：批量整理、标签优化、双链建立、MOC生成
- 任务管理（4个）：创建路书、更新进度、生成报告、任务归档
- 质量检查（3个）：规范检查、文档检查、链接检查

### 方式2：场景示例（推荐进阶用户）⭐⭐

**30分钟搭建体系**：

1. 选择场景：知识管理/技术学习/内容创作/项目协作
2. 复制场景配置到你的知识库
3. 根据实际情况调整规范和标签
4. 使用场景中的Prompt开始工作

### 方式3：自定义搭建（推荐专家用户）⭐

**2-4小时深度定制**：

1. 从规范层开始，逐层搭建
2. 根据需求选择模板组合
3. 创建自己的Prompt库
4. 持续优化形成体系

## ❓ 常见问题

**Q: 必须使用所有模板吗？**

A: 不是。根据需求选择：
- 最小配置：规范层 + 元Prompt库（快速启动）
- 推荐配置：规范层 + 索引层 + 元Prompt库
- 完整配置：四层全部 + 场景示例

**Q: 元Prompt如何使用？**

A: 三步骤：
1. 打开 `04-元Prompt库/` 选择Prompt
2. 复制其中的"Prompt模板"部分
3. 粘贴到AI对话框，填写参数

**Q: 可以修改模板吗？**

A: 可以，而且应该修改！根据场景调整规范、标签、工作流，创建自己的Prompt。

**Q: 旧版模板还能用吗？**

A: 可以！`文档整理/` 目录保留了v1.x旧版，完全兼容。建议逐步迁移到新版。

**Q: 如何自定义Prompt？**

A: 复制现有Prompt模板，修改YAML头部和内容，参考 `04-元Prompt库/` 中的任意Prompt。

## 💡 使用建议

### 新手建议

1. **先理解再使用**
   - 阅读[核心概念](../docs/01-核心概念.md)
   - 理解每个模板的用途
   - 不要盲目复制

2. **从小场景开始**
   - 选择10-20个文件试点
   - 只使用核心规范
   - 逐步扩展

3. **保持简洁**
   - 不要过度设计
   - 规范文档<200行
   - 能用就行

### 进阶建议

1. **自定义扩展**
   - 根据场景调整
   - 添加专属规范
   - 开发自动化工具

2. **版本管理**
   - 使用Git管理
   - 记录变更历史
   - 定期备份

3. **持续优化**
   - 根据实践调整
   - 收集反馈
   - 迭代改进
