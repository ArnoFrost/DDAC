# 安全策略 / Security Policy

Languages: [English](SECURITY.en.md) | 中文

## 报告安全/隐私问题
- 请不要在公开 Issue 中披露敏感细节
- 通过邮件或私信向维护者报告（建议附最小复现与影响范围）
- 在问题确认与修复前，请勿公开披露

## 响应流程
1. 确认问题与影响范围
2. 制定修复方案与时间表
3. 发布修复并更新文档（若涉及）
4. 在 `CHANGELOG.md` 标注安全修复

## 范围说明
- 本仓库为文档与模板为主，不包含执行型代码
- 重点关注：隐私信息泄漏、示例/截图合规、链接范围合规（仅 `docs/*`、`templates/*`）

感谢你为社区安全所做的贡献。
